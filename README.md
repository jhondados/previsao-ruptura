# 🚚 Previsão de Ruptura de Estoque

> Previsão de ruptura de estoque (stockout) com ML. Alerta antes da falta, recomenda reposição e protege vendas e nível de serviço.

**🌐 Demo ao vivo (web app interativo):** https://jhondados.github.io/previsao-ruptura/

[![Vertical](https://img.shields.io/badge/Vertical-Logística-blue)]() [![IA](https://img.shields.io/badge/IA-ML-green)]() [![Demo](https://img.shields.io/badge/Demo-Interativa-orange)]()

## 📊 Resultados-chave (KPIs)
**-41%** Ruptura · **7 dias** Antecip. · **94%** Recall · **R$ 18M** Vendas salvas

## 🧠 Tecnologias de IA
ML · Forecasting · Classification

## ✨ O que faz
- Previsão de stockout por SKU
- Alerta com 7 dias
- Recomendação de reposição
- Proteção de vendas

## 🗄️ Fonte de dados real
[WMS + vendas](https://www.gov.br/mdic)

## ⚙️ Como funciona
1. **Coleta de dados reais** — integração com WMS + vendas
2. **Processamento com IA** — ML · Forecasting · Classification
3. **Dashboard & insights** — KPIs, gráficos e recomendações acionáveis
4. **Ação automatizada** — alertas, predições e decisões em tempo real
